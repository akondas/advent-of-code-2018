# Advent of Code 2018

Solutions to the 2018 Advent of Code challenges in PHP.

To run solve specific day you can use:

```bash
bin/console day:<number> <input>
```

* [Day 1: Chronal Calibration](src/day-01/)

## Tests

To run test suite:

```bash
composer tests
```
